### Railway Ticket Booking System (C)

A console-based Railway Ticket Booking System developed in C using Linked Lists and Queues to simulate reservation, cancellation, and waitlist management.



#### Features

* Book railway tickets
* Cancel confirmed tickets
* Automatically movement to confirmed list when seats free up
* Multiple train support
* Seat allocation using linked lists
* Waitlist management using queues
* Clean, menu-driven interface



#### Data Structures Used

* Singly Linked List → confirmed bookings
* Queue → waiting list



#### Concepts Practiced

* Dynamic Memory Allocation
* Structures
* Pointers
* Linked Lists
* Queue Operations



#### Sample Menu

Railway Ticket Booking System

1. Book Ticket
2. Cancel Ticket
3. Display Bookings
4. Exit



#### Requirements

* GCC compiler (or any C compiler)
* Terminal / Command Prompt



#### Compilation and Execution

For Linux/macOS:

gcc src/railway\_ticket\_booking.c -o railway

./railway
